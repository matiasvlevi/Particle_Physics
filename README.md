

# Pixel Fluid Simulation
##### 2019-04-03
Pygame program with simple particle physics & different elements. 

##### Elements:
* RCK (rock)
* SND (sand)
* DRT (dirt)
* WTR (water)
* WLL (wall)

##### Physiscs:
* Gravity (RCK,SND,DRT,WTR)
* Fluid simulation (WTR)
* Static (WLL)



### Requirements
* pygame


### Usage

Run one of the two batch files.
